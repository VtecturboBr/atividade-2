 📌 Atividade 2

Este repositório contém a implementação da **Atividade 2**, desenvolvida como parte dos estudos em programação.

 🚀 Objetivo

O objetivo deste projeto é aplicar conceitos fundamentais de desenvolvimento, como:

- Estruturação de código
- Lógica de programação
- Manipulação de dados/entrada do usuário
- Boas práticas de organização

 🛠️ Tecnologias utilizadas

- HTML / CSS / JavaScript
- Git e GitHub para versionamento
